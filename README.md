## About The Project
NTNU_Sleepy is a project aimed at creating an app for people experiencing sleep disorders. The app consists of a 4-week training program consisting of different training modules, and offers an alternative to traditional Cognitive Behavioural Therapy for Insomnia. The goal is to make sleep guidance more accessible, as well as providing a platform where the user data can be used for research of sleep related health issues.
